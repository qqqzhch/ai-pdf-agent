# AI PDF Agent - 5 人团队工作流程

> **项目：** AI PDF Agent（第一阶段）
> **团队规模：** 5 人
> **开发方式：** 测试驱动开发（TDD）
> **创建日期：** 2026-03-03

---

## 👥 角色分配

### 5 人团队（建议）

| 角色 | 主要职责 | 次要技能 |
|------|---------|----------|
| **架构师 + 前端开发** | 架构设计 + 代码实现 | Python 能力、架构设计 |
| **后端开发** | 核心功能实现 + 测试 | Python + pytest |
| **前端/CLI 开发** | CLI 框架 + 命令实现 | Python + Click |
| **测试工程师** | 测试用例 + 覆盖率提升 | pytest + mock + pytest-cov |
| **文档工程师** | 文档编写 + 版本管理 | Markdown + Git |

> **注意：** 如果只有 5 人，可以 1 人承担 2 个角色，或者根据实际情况调整
